宇聯商城 — 綠界超商物流測試表

版本：v1.1 ／ 日期：2026-04-16 ／ 環境：正式站（Railway）

第 0 區：前置環境檢查（所有測試開始前必須全部通過）

⚠️ 本區任一項目未通過，請先解決再進行後續測試。

項目 檢查說明 通過條件 ✓
0-1 綠界後台確認「超商取貨」服務已開通
路徑：物流服務 → 廠商後台 → 服務設定 → 超商取貨		 狀態顯示「已開通」，可截圖附檔存證
0-2 Railway 已設定以下三個物流專用環境變數
ECPAY_LOGISTICS_MERCHANT_ID
ECPAY_LOGISTICS_HASH_KEY
ECPAY_LOGISTICS_HASH_IV
注意：物流金鑰與金流金鑰為兩套，不可共用 		Railway Variables 頁面三個變數皆存在且值不為空白
0-3 Railway 最新部署狀態為「Success」
路徑：Railway 專案 → Deployments 分頁		 最新一筆 Deployment 狀態為綠色 Success
0-4 結帳頁選超商取貨，點「選擇門市」能成功開啟綠界地圖視窗 視窗正常開啟，不出現「找不到加密金鑰，請確認是否有申請開通此物流方式」錯誤訊息

第 1 區：結帳頁超商取貨選項顯示

項目 測試步驟 預期結果 ✓
1-1 進入 /shop/checkout，選擇「超商取貨」配送方式 頁面顯示 7-ELEVEN、全家、萊爾富 三個超商選項
1-2 選擇任一超商後，確認頁面狀態 出現「選擇門市」按鈕，尚未選店時顯示提示文字

第 2 區：超商門市選擇（電子地圖）

⚠️ 若第 0 區 0-4 項失敗，跳過本區全部測試，先解決環境變數問題（設定 ECPAY_LOGISTICS_* 三個 Railway 環境變數）後再回來執行。

項目 測試步驟 預期結果 ✓
2-1 選 7-ELEVEN，點「選擇門市」 綠界電子地圖視窗開啟，顯示 7-ELEVEN 門市地圖
2-2 在地圖上選擇一間 7-ELEVEN 門市並確認 地圖視窗關閉，結帳頁顯示所選門市名稱與門市代號
2-3 選 全家，點「選擇門市」 綠界電子地圖視窗開啟，顯示全家門市地圖
2-4 在地圖上選擇一間全家門市並確認 地圖視窗關閉，結帳頁顯示所選門市名稱與門市代號
2-5 選 萊爾富，點「選擇門市」 綠界電子地圖視窗開啟，顯示萊爾富門市地圖
2-6 在地圖上選擇一間萊爾富門市並確認 地圖視窗關閉，結帳頁顯示所選門市名稱與門市代號

第 3 區：超商取貨下單流程

項目 測試步驟 預期結果 ✓
3-1 選好門市後填寫收件人姓名、電話，點「送出訂單」 訂單成功建立，頁面跳轉至訂單確認頁，顯示訂單編號
3-2 確認後台訂單管理頁，找到剛建立的訂單 訂單顯示配送方式為超商取貨，門市資訊正確
3-3 確認綠界物流後台是否收到物流訂單（可選） 綠界物流後台出現對應的物流訂單
