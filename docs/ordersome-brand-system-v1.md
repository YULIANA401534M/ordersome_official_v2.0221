# OrderSome Brand System v1

適用範圍：
- `client/src/pages/brand/*`
- `client/src/components/layout/BrandHeader.tsx`
- `client/src/components/layout/BrandFooter.tsx`
- `client/src/pages/shop/*` 中會直接接觸品牌消費者的前台頁面

目的：
- 讓「來點什麼」的前台設計不再靠單頁發揮
- 建立可跨首頁 / 菜單 / 門市 / 加盟 / 商城沿用的品牌語彙
- 後續所有前台視覺改版先對齊這份記憶，再進入實作

## 1. 品牌定位

來點什麼不是純韓式餐飲，也不是傳統台式早餐店。

目前較準確的品牌理解：
- 台韓混血早午餐品牌
- 年輕、活潑、連鎖感明確
- 既有餐飲品牌，也帶有加盟擴張企圖
- 要有社群感、拍照感、話題感，但不能失去餐點可口度

品牌印象關鍵字：
- 年輕
- 韓系
- 明亮
- 俏皮
- 熱鬧
- 有記憶點
- 好吃
- 可加盟

避免走向：
- 太精品冷感
- 太文青留白
- 太幼稚卡通
- 太夜店 / 太潮牌
- 太像一般早餐加盟站

## 2. 核心視覺語彙

來點什麼的視覺不是「極簡」，而是：
- 明亮大膽
- 有節奏感
- 有畫面動勢
- 帶一點貼紙感、角色感、社群感

可以參考的語法：
- 大標題與副標題做強對比
- 不完全對稱的構圖
- 黃色、白色、灰色作主體
- 黑灰色做文字與視覺壓重
- 區塊之間有清楚情緒切換
- 圖片與插畫可以帶漂浮感、貼紙感、懸浮感

## 3. 標準字

角色分工：
- 品牌展示字：`JF-Kamabit`
- 內容與導覽字：`Plus Jakarta Sans` / `PingFang TC`
- 數字與特殊資訊：保留穩定無襯線或 monospace

使用規則：
- 品牌展示字只用在主視覺、品牌標語、少量重點標題
- 中文長句、導覽列、按鈕、卡片說明、菜單資訊不要大量使用展示字
- 頁面主標題可混用，但以可讀性優先

判斷標準：
- 一個畫面裡只允許 1 個真正高戲劇性的展示標題
- 其餘層級全部回到穩定 UI 字體

## 4. 標準色

基礎色盤：
- Brand Yellow：主品牌高光
- Brand Yellow Light：大面積淡色底
- Brand Gray：正文與穩定支撐色
- Brand Gray Light：背景與邊界
- Brand Gray Dark：重點文字與深色區塊
- White：留白與乾淨感

使用邏輯：
- 黃色不是整頁亂鋪，而是視覺發動機
- 白色負責清爽、餐點乾淨、品牌呼吸感
- 灰色負責成熟、平衡、閱讀
- 黑灰色負責壓重與增加食物質感

比例建議：
- 50% 白 / 淺底
- 25% 灰 / 中性色
- 15% 黃色系
- 10% 深色塊與強對比

## 5. 畫布語法

首頁 / 品牌頁畫布要點：
- 要有一個強 hero
- hero 可以深底配亮黃，也可以亮底配單一焦點物件
- 區塊不能每一段都長一樣
- 至少要有 3 種節奏：
  - 強烈主視覺段
  - 清爽內容段
  - 深色或高對比情緒段

區塊建議：
- Hero：品牌情緒、主訴求、CTA
- Signature：品牌代表商品 / 核心特色
- Social Proof：門市、熱賣、社群、加盟優勢
- Conversion：導向菜單、門市、加盟、商城

## 6. 元件語氣

Header：
- 乾淨但不能太無聊
- Logo 要有存在感
- CTA 要比一般導覽更亮眼

Card：
- 偏圓角
- 有輕微漂浮感
- 可用亮色 top border、貼紙標籤、小圖示增加活潑度

Button：
- 主按鈕應該有明確品牌黃存在感
- 不要過度科技感
- 可帶圓角與厚度，但不要塑膠感太重

Image：
- 食物圖要看起來好吃、乾淨、亮
- 比起太多濾鏡，更重視構圖、比例、留白和容器搭配

## 7. 文案語氣

語氣應該是：
- 年輕
- 直接
- 有趣
- 帶一點韓系流行感
- 但不要過度裝可愛

可以：
- 短句
- 明確利益點
- 社群感標語
- 記憶點口號

避免：
- 太企業官腔
- 太直銷式
- 太像一般早餐加盟模板站

## 8. 與 Dayone / 宇聯的區別

來點什麼前台：
- 要更動感
- 可更大膽
- 可以更有圖像情緒與角色感

宇聯前台 / 集團頁：
- 比較穩重
- 比較企業
- 比較策略性

Dayone / 後台：
- 以耐看、清楚、成熟為優先
- 不應直接套用來點什麼前台語法

## 9. 接下來的執行方式

後續若要做來點什麼前台改版，建議固定順序：
1. 先讀這份 `ordersome-brand-system-v1.md`
2. 先做首頁視覺方向板 / moodboard
3. 先定 hero、色彩比例、字體層級
4. 再改首頁
5. 首頁定調後再延伸菜單、門市、加盟、商城

## 10. 本輪結論

這份文件就是「來點什麼」專屬的專案記憶起點。

之後每次談：
- 黃色怎麼用
- 品牌字怎麼用
- 要不要更韓系
- 要不要更像參考站

都先回到這份文件，不再每次從零猜。
